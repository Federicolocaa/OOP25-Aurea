# Aurea Mediocritas

**Aurea Mediocritas** è un simulatore gestionale universitario ispirato alle meccaniche di "Reigns". Il giocatore assume il ruolo di un Rettore e deve bilanciare quattro parametri fondamentali (Finanze, Consenso Studenti, Consenso Docenti, Reputazione Pubblica) per tre anni accademici. Ogni scelta effettuata tramite le carte evento avrà un impatto diretto sul destino dell'ateneo.
